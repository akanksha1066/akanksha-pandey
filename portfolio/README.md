# Akanksha Pandey — Data Analyst Portfolio

A fully responsive, single-page portfolio site built with plain HTML5, CSS3 and vanilla JavaScript — no frameworks or libraries.

## Structure

```
portfolio/
├── index.html
├── css/
│   └── style.css
├── js/
│   └── script.js
├── assets/
│   ├── images/      → profile photo + favicon
│   ├── icons/        → reserved for future icon assets
│   └── resume/       → downloadable resume PDF
└── README.md
```

## Sections

- **Hero** — name, title, short intro, CTA buttons, photo
- **About Me** — summary, "What I Do", quick facts
- **Skills** — categorized badges (languages, analysis tools, BI/reporting)
- **Resume** — education timeline, certifications, achievements, resume download
- **Projects** — the three featured projects with tech stacks
- **Contact** — email, phone, location, social links, front-end contact form

## Notes

- The contact form is front-end only (no backend wired up); it validates input and shows a confirmation message.
- Colors, type and layout were modeled on the supplied reference design (bold black/yellow palette, circular photo, side dot-nav, rounded "about" panel, KPI-style fact circles).
- Update the `LinkedIn` / `GitHub` `href="#"` placeholders in `index.html` with real profile URLs before publishing.
- Replace `assets/images/favicon.png` if you'd like a different favicon.

## Browser support

Tested against modern evergreen browsers: Chrome, Edge, Firefox, Safari.
